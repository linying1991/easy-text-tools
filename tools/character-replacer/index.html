Frequently Asked Questions

How does character replacement work?

The tool finds every occurrence of the specified character in the "Replace Character" field and replaces it with the character in the "With Character" field, throughout your entire input text.

Can I replace multiple different characters at once?

This tool replaces only one character at a time. To replace multiple different characters, you would need to use the tool multiple times or use a more advanced find-and-replace tool.

Is the replacement case-sensitive?

Yes, the replacement is case-sensitive. If you want to replace both uppercase and lowercase versions of a letter, you need to perform separate replacements for each case.

Can I replace a character with nothing (delete it)?

Yes, leave the "With Character" field empty to remove all occurrences of the specified character from your text.
