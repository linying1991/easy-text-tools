Frequently Asked Questions

What's the difference between capitalizing words and sentences?

"First letter of each word" capitalizes the first letter of every word (like Title Case). "First letter of sentences" capitalizes only the first letter after sentence-ending punctuation (. ! ?).

How does word capitalization work?

The tool identifies word boundaries using whitespace and common delimiters, then capitalizes the first letter following these boundaries.

Will this preserve the original formatting?

Yes, the tool preserves all original formatting including spacing, line breaks, and punctuation. Only the case of letters is changed.

Can I use this for titles and headings?

Yes, this is perfect for formatting titles, headings, and other text that requires Title Case capitalization. For sentence case, it's ideal for normal text paragraphs.
