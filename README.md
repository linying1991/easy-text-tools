# Easy Text Tools

A simple, privacy-focused Easy Text Tools that runs entirely in your browser. Extract frames from animated GIFs with no uploads required.

## Features

- Extract frames from animated GIF files
- No file uploads - everything happens locally in your browser
- Multiple frame extraction options:
  - All frames
  - First frame only
  - Last frame only
  - Custom frame ranges (e.g., "1-5" or "1,3,5")
- Custom size options (original, custom width, custom height)
- Batch conversion support for multiple GIF files
- Drag and drop interface
- Multilingual support (English/Chinese)

## Technology

- **gifuct-js**: Advanced GIF parsing and frame extraction
- **Vite**: Fast build tool and development server
- **Pico CSS**: Minimal CSS framework
- **File-saver**: Client-side file downloads
- **fflate**: Fast compression for ZIP files

## Usage

1. Open the website
2. Select frame extraction options (all frames, specific frames, or custom range)
3. Choose output size settings if needed
4. Select GIF files or drag them to the drop zone
5. Download your extracted PNG frames (single PNG or ZIP file)
